## How Markdown works

Regular paragraphs are just text. Leave a blank line between paragraphs.

You can make text **bold** or *italic*. Links look like [this](https://example.com).

> Blockquotes are useful for pulling out key insights or quoting sources.

### Subheadings

Use `##` for major section breaks and `###` for smaller ones. The template styles them differently — `##` gets the serif display font, `###` gets a smaller uppercase treatment.

When you're ready to publish a new piece, you create a new `.md` file in the `src/pages/posts/` folder and follow this same pattern.
