This is another sample post to show how multiple pieces look on the homepage index.

The frontmatter at the top of each file controls the title, subtitle, date, and tag that appear both on the homepage and on the article page itself.

## Adding a new post

1. Create a new `.md` file in `src/pages/posts/`
2. Name it with dashes: `my-article-title.md`
3. Add the frontmatter block at the top (copy from this file)
4. Write your piece below the `---`
5. Push to GitHub — Vercel deploys automatically

That's the entire workflow.
